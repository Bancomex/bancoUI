## Installation
# 1. Clone our project
### Open a terminal and the following command:
```
$ git clone https://github.com/Mixcoatldev/bancoUI.git
```

# 2. Install dependencies
### Open a terminal and the following command:
```
cd bancoUI
npm install
```

## Usage
### Run our project
```
npm start
```
